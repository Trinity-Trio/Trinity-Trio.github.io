# web-tritri

https://trinity-trio.github.io/web/

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# how to deploy
$ yarn deploy
# or npm run deploy
```

For detailed explanation on how things work, checkout [Nuxt.js docs](https://nuxtjs.org).
